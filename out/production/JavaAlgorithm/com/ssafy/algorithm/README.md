1. 문제 꼼꼼히 읽기 -> 읽으면서 어떤 방법으로 풀 지 생각
2. 코드 기획하고 설계하기, 아주 작은 부분까지 디테일하게 정리
3. 해야할 체크리스트 의사코드 형태로 작성하기
4. 의사코드를 코드로 구현하고 체크하기
5. 코드 전체 점검하기
